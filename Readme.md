# 🌳 Fractal Tree Animation with Audio

A beautiful, interactive recursive fractal tree animation built using **HTML5 Canvas** and **JavaScript**. This project is a modern web recreation of the classic Python Turtle graphics, featuring smooth real-time animation and ambient background music.

## ✨ Features

* **Recursive Algorithm:** Uses mathematical recursion to generate complex and natural branch structures.
* **Smooth Animation:** The tree grows branch-by-branch, providing a satisfying visual experience.
* **Audio Integration:** Includes background music that triggers upon starting the animation.
* **Speed Control:** Optimized drawing intervals to ensure the "growing" effect looks natural.
* **Responsive Design:** The canvas dynamically scales to fit any screen resolution.

## 🛠️ Installation & Usage

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/rayansiddiquii/fractal-tree-animation.git](https://github.com/rayansiddiquii/fractal-tree-animation.git)
    ```

2.  **Add your music:**
    Replace an MP3 file named `audio.mp3` in the root directory.

3.  **Run the project:**
    Simply open `index.html` in any modern web browser (Chrome, Edge, Firefox, etc.).

## 📂 Project Structure

* `index.html` - The main structure and entry point for the application.
* `style.css` - Custom styling for a dark, immersive user interface.
* `script.js` - The core logic for recursion, drawing intervals, and animation.

## ⚙️ Technologies Used

* **HTML5 Canvas:** For high-performance 2D rendering.
* **CSS3:** For layout and button styling.
* **JavaScript (ES6+):** For the recursive logic and animation control.
