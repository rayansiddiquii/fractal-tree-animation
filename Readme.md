# 🌳 Fractal Tree Animation with Audio

A beautiful, interactive recursive fractal tree animation built using **HTML5 Canvas** and **JavaScript**. This project is a modern web recreation of the classic Python Turtle graphics, featuring real-time animation and ambient background music.

## ✨ Features
* **Recursive Algorithm:** Uses mathematical recursion to generate complex branch structures.
* **Smooth Animation:** The tree grows branch-by-branch, providing a satisfying visual experience.
* **Audio Integration:** Includes background music that triggers upon starting the animation.
* **Speed Control:** Optimized drawing speed to ensure the "growing" effect looks natural.
* **Responsive Design:** The canvas dynamically scales to fit any screen resolution.


## 🛠️ Installation & Usage
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/rayansiddiquii/fractal-tree-animation.git](https://github.com/rayansiddiquii/fractal-tree-animation.git)
    ```
2.  **Add your music:**
    Relace an MP3 file named `audio.mp3/mpeg` in the root directory.
3.  **Run the project:**
    Simply open `index.html` in your favorite web browser.

## 📂 Project Structure
* `index.html` - Main entry point with the canvas and audio setup.
* `style.css` - Custom styling for a dark, immersive UI.
* `script.js` - Logic for recursion, drawing intervals, and animation frames.

## ⚙️ Technologies Used
* **HTML5 Canvas:** For high-performance 2D rendering.
* **CSS3:** For layout and button styling.
* **JavaScript (ES6+):** For the recursive logic and animation control.
